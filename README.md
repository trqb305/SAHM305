## ✨ الميزات الأساسية

- 🌓 **وضع ليلي وصباحي** كامل مع تبديل لحظي وحفظ الاختيار
- 🇸🇦 **عربي RTL أصيل** بخط Cairo (أوزان خفيفة 400/500/600 فقط)
- ⚡ **سبعة محركات تحليل** مستقلة، كل محرك يعطي درجته منفصلاً
- 📊 **قرار من 100** — ≥75 نفّذ · 55–74 مشروط · 40–54 راقب · أقل رُفض
- 🔒 **خطة مُقفلة (Frozen Plan)** — كل فرصة مختومة بتوقيت النشر
- 🔄 **تحديث تلقائي كل 30 ثانية** بعدّاد دائري مرئي
- 📰 **شريط أخبار ذكي** بثلاث مستويات (هادئ/حذر/خطر)
- 👑 **لوحة مدير كاملة** مع تبديل الأدوار للمعاينة
- 💎 **نظام باقات أربعة** (Radar/Signal/Edge/Alpha)
- 📱 **متجاوب بالكامل** للجوال والتابلت والديسكتوب

## 🚀 البدء السريع

### 1. تثبيت المتطلبات

تأكد من تثبيت **Node.js 18.17+** و **npm** أو **pnpm**.

```bash
node -v   # >= 18.17
npm -v
```

### 2. تثبيت الحزم

```bash
npm install
```

### 3. إعداد متغيرات البيئة

```bash
cp .env.example .env.local
```

ثم عدّل `.env.local` بالقيم الفعلية.

### 4. تشغيل خادم التطوير

```bash
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000)

- **البريد:** `nif305@gmail.com`

أي بريد وكلمة مرور أخرى ستدخل كمستخدم عادي (للمعاينة فقط).

## 🎨 نظام التصميم

### الألوان

النظام مبني على **CSS Variables** مع وضعين:

| المتغير | الوضع الليلي | الوضع الصباحي |
|---|---|---|
| `--bg-deepest` | `#060D14` | `#FAF7F0` |
| `--bg-card` | `#0F1822` | `#FFFFFF` |
| `--text-primary` | `#E6ECF3` | `#1C2432` |
| `--gold` | `#C9943A` | `#B28226` |
| `--green` | `#00C896` | `#00A87C` |
| `--red` | `#F04760` | `#D7334B` |

### الخطوط

- **العربي:** Cairo (أوزان 400, 500, 600 فقط — لا أوزان غامقة)
- **الأرقام:** JetBrains Mono (مع `tabular-nums`)
- **الاتجاه:** RTL للنص، LTR للأرقام والأسعار
